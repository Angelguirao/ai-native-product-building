## Clarity (must pass)

- [ ] Core task completable without instruction
- [ ] Primary action obvious on first screen
- [ ] Copy uses user's words (from discovery), not internal jargon
- [ ] Error and empty states exist — not happy-path only

## Character (must pass for trust/identity products)

- [ ] Matches design intent axis (warm/clinical, dense/airy, etc.)
- [ ] Consistent with token set — no rogue fonts or colors
- [ ] Feels intentional for target audience — not generic SaaS

## Craft (nice to have pre-launch)

- [ ] Typography hierarchy readable on mobile
- [ ] Motion has purpose — or none
- [ ] Loading states don't flash layout
